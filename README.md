# palindromes

my goal was to find an algorithmic way of finding these
|   |   |   |   |   |
|:-:|:-:|:-:|:-:|:-:|
| R | O | T | A | S |
| O | P | E | R | A |
| T | E | N | E | T |
| A | R | E | P | A |
| S | A | T | O | R |

in order to do that, we need to parse the dictionary to find *palindromes* and *heteropalindromes* of the requisite length
